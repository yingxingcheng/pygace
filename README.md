# pygace

#### Description
This project is used to search ground state of crystal structure using genetic algorithms based 
cluster expansion methods.

#### Installation

For Users
1. git clone https://yingxingcheng/pygace.git 
2. cd pygace
3. pip install 
   
For developer
1. git clone https://yingxingcheng/pygace.git 
2. cd pygace 
3. pip install -e .
